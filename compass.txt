安装compass
$ gem update --system
$ gem install compass
创建项目
$ compass create <myproject> --syntax sass
已有项目
$ cd <myproject>
$ compass install compass 
单个文件
$ sass --watch style.scss:style.css
整目录
$ sass --watch stylesheets/sass:stylesheets/css

更新
gem update compass compass
